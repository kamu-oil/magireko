>> 弱チェ間カウント
   朝一0G・弱チェ引いたGを0Gとして、次Gから1G,2G,3Gと数える。
   弱チェを引いたら、最上部の白囲いをタップ。
   数えている時点のG数を入力し、記録ボタンを押す。
   また0Gに戻って、1G,2G,3Gと数える。

>> ボーナス比率カウント
   BBかRBが揃ったG数をカウンターで確認する。
   250G以内なら左下、251G以上なら右下のボタンををタップ。

>> 結果画面
   赤太文字になったところは注目ポイント。
   設定推測と、弱チェ・ボーナスが設定3以上キープまたは上振れ中。

>> データ保存・読込
   記録ボタンの左にある、保存と読込でセーブ・ロード。
   現在の内容をクリアするには、ページ更新またはブラウザを終了。
