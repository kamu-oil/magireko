■弱チェ間カウント<br>
　遊技開始・弱チェを引いたGを0Gとして、次Gから1G,2G,3Gと数える。<br>
　弱チェを引いたら、最上部の白囲いをタップ。<br>
　数えている時点のG数を入力し、記録ボタンを押す。<br>
　また0Gに戻って、1G,2G,3Gと数える。<br>
<br>
■ボーナス比率カウント<br>
　BBかRBが揃ったG数をデータカウンターで確認する。<br>
　250G以内なら左下、251G以上なら右下のボタンををタップ。<br>
<br>
■結果画面<br>
　赤太文字は注目ポイント。<br>
　設定推測と、弱チェ・ボーナスが設定3以上キープまたは上振れ中。<br>
<br>
■データ復元<br>
　ページ更新・ブラウザ終了でデータクリア。<br>
　自動保存するので、通常は復元ボタンで再開可能。<br>
　ブラウザの設定等により復元できなくなる場合あり。
